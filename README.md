# Developer Workspace & Assets

This repository serves two primary functions for my workflow:

### 1. Asset Hosting / Content Delivery Network (CDN)
It acts as a personal content delivery network to host public images, static assets, and profile pictures used across my developer portfolios (for example, freeCodeCamp) and project documentation. 

### 2. Project Notes & Documentation
It also functions as a digital notepad. I use this space to store technical documentation, log ideas, and keep track of logic snippets for my projects so they are always accessible in one central location.
